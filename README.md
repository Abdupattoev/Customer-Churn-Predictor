# Customer Churn Prediction App

End-to-end machine learning project predicting telecom customer churn.

## Live App
(https://customer-churn-predictor-sbbqi5nhhslr2v7y5puvrm.streamlit.app)

## Problem
Customer churn reduces company revenue.  
This model predicts which customers are likely to leave so companies can act early.

## Model
- Random Forest Classifier
- Accuracy: ~80%
- Dataset: IBM Telco Customer Churn

## Tech Stack
Python, Pandas, Scikit-learn, Streamlit

## Features
- Interactive prediction app
- Real-time churn probability
- Business-focused insights
